# Contributing to WC2026 ML Predictor

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, open a new issue, including as much information as possible.

### Suggesting Enhancements
* Open a new issue with a clear title and description of the enhancement.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the project's style.
3. Issue a pull request!

## Code of Conduct
This project and everyone participating in it is governed by the [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
